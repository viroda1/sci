# Science Lab Toolkit - VIRO SCI

A stunning science tools website continuing the purple and black halftone design language. Features interactive thermometers, pH scale, measurement converters, and essential lab equipment visualizations.

## ✨ Features

- **Interactive Thermometer** - Click to adjust temperature from -10°C to 110°C with visual mercury column
- **pH Scale Indicator** - Interactive slider showing acidity/alkalinity with color-coded feedback
- **Unit Converter** - Convert between Celsius, Fahrenheit, and Kelvin with real-time calculations
- **Lab Equipment Cards** - Beakers, microscopes, and other essential tools
- **Measurement Tools** - Mass, volume, and length converters with visual indicators
- **Modern Design** - Consistent purple/black scheme with halftone patterns and glass effects

## 🎨 Design Highlights

- **Halftone Textures** - Layered radial gradient dot patterns throughout
- **Interactive Elements** - Sliders, buttons, and real-time updates
- **Scientific Styling** - Lab-inspired visual elements with glowing accents
- **Responsive Layout** - Adapts beautifully to all screen sizes
- **Purple Gradient Palette** - Rich purples from deep black to bright lavender

## 🛠️ Technologies Used

## 💡 How to Use

### Thermometer
- Click the up/down arrows to adjust temperature
- Watch the mercury column rise and fall
- Temperature displays in real-time
- Color changes from blue (cold) to red (hot)

### pH Scale
- Drag the slider or use +/- buttons
- See color change from red (acidic) to purple (alkaline)
- Real-time pH value and description update
- Visual indicator shows acid/base/neutral zones

### Unit Converter
- Select conversion type (temp, mass, volume, length)
- Enter values in any field
- All related units update automatically
- Precision to 2 decimal places

### Lab Equipment
- Hover over cards for glow effects
- Visual representations of common lab tools
- Quick-reference measurements

## 🧪 Scientific Tools Included

- **Temperature**: Celsius, Fahrenheit, Kelvin converter
